Upgrade styles and axes

Supporting a variety of related styles.

[Explaining the workings of interpolation, multiple master axes and variable fonts. There is a number of slides of previous presentations that can be used as illustrations here.]

Gerrit Noordzij’s cube is based on a merge between expansion contrast (sharp nib pencil) and translation contrast (broad nib pencil). The type of contrast used in Upgrade is based on translation, with little difference between vertical thick and horizontal thin strokes. The animation (based on software by Just van Rossum’s Daily Drawbot) shows where the contrast principle of Upgrade is located in Noordzij’s cube.
Upgrade styles in this example are respectively: Light, Book, Regular, Semibold and Black.

Different drawings for different weights

Depending on the available space of stems and counts in the various weights, some characters have different drawings. There is no room for the vertical bar of dollar and cent to strike completely through. Note that although all weights are shown here (ranging from Hairline to UltraBlack), details in the extreme weights may not make them the best candidate to use them in this size.

More variations in the near future releases

Soon more variations in the Upgrade family will be released, such as Upgrade Condensed, Upgrade Wide and Upgrade Text. And these variant will be combined, together with other parametric axes, in Upgrade Var. All these are intended to increase typographic quality for specific usage and size.

Alternative styles, using the Upgrade family as a template

Also planned is a number of variants that cannot easily be expressed in interpolating axis, needed for Variable Fonts. They have a more stylistic approach based on specific usage, where readibility it not the first priority. Projected example labels of these non-variable families: Upgrade Round, Upgrade Notouch, Upgrade Logo, Upgrade Ink and Upgrade Metal.

Axis examples

  • Weight (wght)

    The default weight axis of Upgrade is made from 5 masters, to cover the extreme range between Upgrade Hairline and Upgrade UltraBlack. Click on the animation to view a similar document as PDF.

  • How this animation was made

    This animated gif of 8 frames was made by this PageBot script, working as follows:

    [1] Start a script in DrawBot + PageBot which takes a random selection from a list of words and then combines them in pairs. Each pair will fill a line.
    [2] If the number of characters in a line is smaller than 12 characters – or otherwise with a chance of 10% – convert the line to capitals.
    [3] The stack of lines start on top of a page: put the cursor at the top-left corner of the margin.
    [4] Select a random weight from the TYPETR Upgrade family. There is a 20% chance that italics are selected.
    [5] Then calculate the point size needed, to make the line fit on the document page width.
    [6] Calculate the enclosing pixel box and move the cursor position vertical to the next line, increased by some leading.
    [7] Repeat this until the page is filled.
    [8] Then repeat this page squence for the required number of frames in the animation.
    [9] Save as PDF and as animated gif, with frame duration of 3 seconds.

  • Baseline-J [ss08] feature

    Special note that the capital lines are using the OT-feature [ss08], which forces Upgrade to use the inline-J instead of the standard hanging-J. This creates a closely wrapped bounding box on capital words.
