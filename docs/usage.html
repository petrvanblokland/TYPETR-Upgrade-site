Upgrade documents

  • Business cards

    As part of a larger identity – or just as single items – the typography of business cards benefits from Upgrade’s variety of typographic options. Click on the image to view them as PDF document.

  • How these card were made

    A DrawBot/PageBot script created the documents and saved them as PNG and PDF files. In pseudo code this works as follows:

    • Start a script in DrawBot + PageBot and create a page of standard business card size for the intended country. • Accept name, function and additional address information directly from code, from a file or from a database source.
    • Create styles with their typographic values for each of the different text functions on the card.
    • Combine the information with their related style.
    • Select a random weight from the TYPETR Upgrade family for the “logo”. In reality this would be define by branding instructions.
    • Then calculate the point size and tracking needed, to make the line fit on the width of the card.
    • Then repeat this squence for all input names.
    • Save as PDF and PNG files.
