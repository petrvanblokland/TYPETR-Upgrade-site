Scripting with PageBot

For users with existing scripting experiences: all examples of this site are generated by PageBot, an Open Source scripting library based on DrawBot and Python. These scripts can be downloaded here. (The Upgrade fonts used in the examples are not included. Various types of licenses are available here).
